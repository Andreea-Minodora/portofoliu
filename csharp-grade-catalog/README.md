
# Catalog de Note – Aplicație C# (Windows Forms)

## Descrierea proiectului

Acest proiect constă într-o **aplicație desktop realizată în C# folosind Windows Forms**, care permite gestionarea unui **catalog de note pentru studenți**.

Aplicația oferă funcționalități pentru administrarea studenților, disciplinelor și notelor, folosind o bază de date **SQL Server** pentru stocarea informațiilor.

Interfața aplicației este organizată în mai multe formulare care permit utilizatorului să gestioneze datele într-un mod intuitiv.

---

## Funcționalități

Aplicația permite:

* administrarea studenților (adăugare, modificare, ștergere)
* administrarea disciplinelor
* introducerea și gestionarea notelor
* filtrarea datelor după facultate, specializare și student
* calcularea **mediei generale**
* afișarea situației studentului (**promovat / nepromovat**)
* exportarea datelor în **format CSV**

---

## Modulele aplicației

### Modulul Studenți

Permite gestionarea informațiilor despre studenți:

* introducerea datelor personale
* actualizarea datelor
* ștergerea studenților
* căutarea studenților după prenume

### Modulul Discipline

Permite administrarea disciplinelor:

* adăugarea disciplinelor
* modificarea informațiilor despre disciplină
* ștergerea disciplinelor
* filtrarea disciplinelor după facultate și specializare

### Modulul Catalog

Permite gestionarea notelor studenților:

* adăugarea notelor
* modificarea și ștergerea notelor
* calcularea mediei
* afișarea situației studentului
* exportarea datelor în CSV

---

Acest proiect a fost realizat pentru a exersa:
