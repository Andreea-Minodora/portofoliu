
# 📊 Sales Dashboard (Python + Pandas + Streamlit)

## 🚀 Descriere proiect

Acesta este primul meu proiect de analiză de date realizat în Python.
Proiectul are ca scop explorarea și analiza unui set de date de vânzări folosind Pandas și vizualizarea rezultatelor printr-un dashboard interactiv construit cu Streamlit.

---

## 🎯 Ce am realizat

În acest proiect am implementat:

* calculul venitului total (Total Revenue)
* valoarea medie a comenzilor (Average Order Value)
* numărul total de comenzi
* identificarea celor mai vândute produse
* identificarea celor mai valoroși clienți
* analiză pe categorii de produse
* dashboard interactiv cu filtru pe categorie

---

## 🛠️ Tehnologii folosite

* Python
* Pandas
* Streamlit
* Matplotlib (pentru grafice în analiza inițială)

---

## 📂 Structura proiectului

* `data.csv` – setul de date
* `analysis.py` – analiza datelor în Python
* `dashboard.py` – dashboard interactiv
* `plots/` – grafice generate

---

## ▶️ Cum rulezi proiectul

1. Instalează dependențele:

```bash
pip install pandas streamlit matplotlib
```

2. Rulează dashboard-ul:

```bash
python -m streamlit run dashboard.py
```
