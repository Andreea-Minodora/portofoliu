# Proiecte

## 1. Analiza accidentelor rutiere (Power BI)

Acest proiect constă într-un dashboard interactiv realizat în Power BI pentru analiza datelor despre accidente rutiere din Marea Britanie.

Funcționalități:

* analiza accidentelor în funcție de lună
* analiza accidentelor în funcție de ziua săptămânii
* distribuția severității accidentelor (Slight, Serious, Fatal)
* vizualizarea accidentelor pe hartă
* filtrarea datelor în funcție de an

Tehnologii utilizate:

* Power BI
* Analiză exploratorie a datelor (EDA)
* Vizualizare de date
