
# 📊 Student Grades Dashboard (Python)

## 🚀 Despre proiect

Acest proiect reprezintă un dashboard interactiv realizat în Python folosind Streamlit, pentru analiza notelor studenților.

Scopul proiectului este de a explora datele și de a obține insight-uri relevante despre performanța studenților.

---

## 🎯 Funcționalități

* 📈 Media generală a notelor
* ➕ Suma notelor
* 🔍 Filtru interactiv pe curs
* 📊 Media notelor pentru fiecare student (grafic)
* 🏆 Identificarea studentului cu cea mai mare medie
* 📋 Vizualizarea datelor filtrate

---

## 🛠️ Tehnologii utilizate

* Python
* Pandas
* Streamlit

---

## 📂 Structura proiectului

* `students.csv` – dataset
* `dashboard.py` – aplicația Streamlit
* `(python-students-dashboard-analysys/image.png` – screenshot dashboard

---

## ▶️ Rulare

```bash
python -m streamlit run dashboard.py
```
