# Finance Dashboard – UI Design (Figma)

## Descrierea proiectului

Acest proiect constă într-un **design de dashboard financiar realizat în Figma**, care prezintă indicatori financiari într-o interfață modernă și intuitivă.

Scopul proiectului a fost realizarea unui **dashboard UI pentru o aplicație financiară**, care să permită utilizatorilor să vizualizeze rapid informații importante precum veniturile, cheltuielile, economiile și evoluția financiară.

Designul urmărește o structură clară și o experiență vizuală ușor de utilizat.

---

## Funcționalități ale designului

Interfața include următoarele componente:

* **Sidebar de navigare** pentru acces rapid la secțiuni (Dashboard, Transactions, Reports, Goals, Settings)
* **Carduri cu indicatori financiari**:

  * Total Income
  * Total Expenses
  * Savings
  * Growth
* **Grafic de evoluție a veniturilor** pentru ultimele luni
* **Diagramă circulară** pentru distribuția cheltuielilor
* **Bară de căutare** pentru navigare rapidă
* **Design modern cu temă dark**

---

## Instrumente utilizate

* **Figma**
* **UI Design**
* **Dashboard Layout Design**

---


## Scopul proiectului

Acest proiect a fost realizat pentru a exersa **designul de interfețe pentru dashboard-uri financiare** și organizarea informațiilor într-un mod vizual clar și eficient.
