
# Website Hotel – HTML & CSS

## Descrierea proiectului

Acest proiect constă într-un **site web pentru prezentarea unui hotel**, realizat folosind **HTML și CSS** și dezvoltat local cu ajutorul **XAMPP**.

Site-ul prezintă informații despre hotel, tipurile de camere disponibile, serviciile oferite și include un formular pentru realizarea unei rezervări online.

Scopul proiectului a fost dezvoltarea unei interfețe web funcționale și intuitive, utilizând tehnologii de bază pentru dezvoltarea frontend.

---

## Funcționalități

Site-ul include mai multe secțiuni și pagini:

* **Pagină principală** cu prezentarea hotelului
* **Descriere hotel** și facilități
* **Servicii și prețuri pentru camere**
* **Restaurant și facilități**
* **Formular de rezervare online**
* **Review-uri ale clienților**
* **Navigare între pagini folosind meniul principal**

---

## Tehnologii utilizate

* **HTML**
* **CSS**
* **XAMPP** (pentru rularea serverului local)

---



## Scopul proiectului

Acest proiect a fost realizat pentru a exersa dezvoltarea unui **site web cu mai multe pagini**, organizarea conținutului și realizarea unui layout modern folosind HTML și CSS.
